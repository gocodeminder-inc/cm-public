# CodeMinder — Engineering Intelligence Platform

AI-powered triage for engineering data. Upload logs, crash reports, scanner outputs, customer feedback and more — get deduplicated, prioritized action items linked to business impact.

## What It Does

- Ingests raw engineering data (logs, crash reports, code and config scanners, support tickets and more ...)
- Classifies, deduplicates, and scores issues using AI
- Surfaces prioritized action items with root cause analysis and business impact
- Learns from your feedback to improve over time

## Stack

React · TypeScript · FastAPI · PostgreSQL (RDS) . PyTorch
